
RADIO PWA — GitHub Pages
========================
1) Subí TODO el contenido de este ZIP a tu repo bajo / (raíz) o a una carpeta (ej: /radio-daniel).
2) Activá GitHub Pages en Settings → Pages → Source: Deploy from a branch → main/(root) o main/docs según dónde lo subas.
3) Abrí: https://<tu-usuario>.github.io/<tu-repo>/ o /<subcarpeta>/
4) Si actualizás archivos, forzá refresh del SW: Menú del navegador → Limpiar datos/Recargar duro (o cambia la versión del cache en sw.js).
5) Streams: si no suena alguna estación puede ser por CORS o códec (AAC en algunos navegadores de escritorio).
